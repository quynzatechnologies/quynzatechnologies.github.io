Módulo 1.3. Conceptos básicos de programación

La programación de computadores es el proceso de escribir instrucciones o código para indicarle a un computador cómo realizar una tarea específica. El conjunto de instrucciones que ejecuta una tarea se conoce como algoritmo, que es un conjunto definido, ordenado y no ambiguo de pasos para resolver un problema o realizar un cómputo.

Antes de entender la programación, es fundamental conocer la estructura básica de un computador. Este está compuesto por una unidad de proceso (CPU, microprocesador o microcontrolador), una memoria principal (RAM, ROM, memoria Flash) y un conjunto de periféricos que le permiten interactuar con el entorno (teclado, pantalla, mouse, parlantes, sensores y actuadores, entre otros). Los datos e instrucciones se almacenan en la memoria y son transferidos por los buses hacia la unidad de proceso para su ejecución.

En la programación, todas las instrucciones se organizan usando estructuras básicas:

  • Secuencia: ejecución ordenada de instrucciones, una tras otra.
  • Selección: bifurcación condicional que ejecuta uno u otro bloque según una condición lógica.
  • Ciclo: repetición de un bloque de instrucciones mientras se cumpla una condición.

El computador interpreta estas instrucciones en lenguaje de máquina, compuesto por 0s y 1s. Dado que es inviable para un humano programar directamente en este lenguaje, se utilizan lenguajes de programación que deben traducirse a binario. Esta traducción puede hacerse mediante:

  • Compiladores: traducen todo el programa antes de su ejecución (ej. Java, C++).
  • Intérpretes: traducen y ejecutan instrucción por instrucción (ej. Python, JavaScript).

En este curso, utilizaremos Python y su versión optimizada para microcontroladores, MicroPython, que son lenguajes interpretados, lo que permite probar y depurar el código de forma interactiva.
