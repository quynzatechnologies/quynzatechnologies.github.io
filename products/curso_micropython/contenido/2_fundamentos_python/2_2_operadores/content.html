Módulo 2.2. Operadores y expresiones

En Python, un operador es un símbolo especial que indica la realización de una operación sobre uno o más valores (operandos). Una expresión es la combinación de variables, valores y operadores que produce un resultado.

  1. Operadores aritméticos

    Se usan para realizar cálculos matemáticos:

    Operador Descripción Ejemplo Resultado
    + Suma 5+3 8
    - Resta 5-3 2
    * Multiplicación 5*3 15
    / División (resultado flotante) 5/2 2.5
    // División entera 5//2 2
    % Módulo (resto) 5%2 1
    ** Potencia 5**2 25

  2. Operadores de comparación

    Sirven para comparar valores y devuelven un resultado booleano (True o False):

    Operador Descripción Ejemplo Resultado
    == Igual que 5 == 3 False
    != Distinto de 5 != 3 True
    > Mayor que 5 > 3 True
    < Menor que 5 < 3 False
    >= Mayor o igual que 5 >= 5 True
    <= Menor o igual que 3 <= 5 True

  3. Operadores lógicos

    Se utilizan para combinar condiciones:

    Operador Descripción Ejemplo Resultado
    and Verdadero si ambas condiciones son verdaderas (5 > 3) and (2 < 4) True
    or Verdadero si al menos una condición es verdadera (5 > 3) or (2 > 4) True
    not Invierte el valor lógico not (5 > 3) False

  4. Operadores de asignación

    Permiten asignar valores a variables y, en algunos casos, modificarlos directamente:

    Operador Ejemplo Equivalente a
    = x = 5 Asignar valor 5 a x
    += x += 3 x = x + 3
    -= x -= 2 x = x - 2
    *= x *= 4 x = x * 4
    /= x /= 2 x = x / 2

  5. Otros operadores útiles

    • Concatenación de cadenas (+):
      mensaje = "Hola" + " Mundo" print(mensaje)
    • Repetición de cadenas (*):
      eco = "ESP32 " * 3 print(eco)
    • Operadores de pertenencia (in, not in):
      if "rojo" in ["rojo", "azul"]: print("Color encontrado")
    • Operadores a nivel de bits: permiten operaciones lógicas sobre bits individuales (&, |, ^, ~, <<, >>).

  6. Aplicaciones con ESP32

    En proyectos con MicroPython y ESP32, los operadores no solo se usan para cálculos, sino también para evaluar condiciones en el control de sensores y actuadores, por ejemplo:
    temperatura = 38 humedad = 45 if temperatura > 30 and humedad < 50: print("Encender ventilador")
    Este tipo de expresiones es esencial para condicionar el comportamiento del ESP32 según datos de sensores.
