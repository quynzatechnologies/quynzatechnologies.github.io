Ejercicio 2-2: Tabla de multiplicar del 8 con ciclo while

Objetivo. Practicar el uso de estructuras de control repetitivas (while). Generar una tabla de multiplicar de forma automática.

Procedimiento.

  1. Abrir Thonny y crear un nuevo archivo .py.
  2. Definir la variable i con valor inicial de 1.
  3. Usar un ciclo while que se repita mientras i sea menor que 10.
  4. Dentro del ciclo, imprimir la multiplicación de 8 por i.
  5. Incrementar el valor de i en 1 en cada iteración para avanzar en la tabla.

# Tabla de multiplicar del 8
i = 1
while i < 10:
    print(" 8 X ", i, " = ", 8 * i)
    i = i + 1

Actividades sugeridas.

  • Modificar el número base (8) para generar la tabla de otro número.
  • Cambiar el límite del ciclo para mostrar más o menos multiplicaciones.
  • Reemplazar el ciclo while por un ciclo for y comparar resultados.

Experimentos.

  • Solicitar al usuario el número para generar la tabla en lugar de usar el 8.
  • Mostrar las tablas del 1 al 10 usando un ciclo anidado.
