Ejercicio 2-3: Tablas de multiplicar del 1 al 10 con ciclos while anidados

Objetivo. Practicar el uso de ciclos while anidados. Generar de forma automática las tablas de multiplicar del 1 al 10.

Procedimiento

  1. Abrir Thonny y crear un nuevo archivo .py.
  2. Definir una variable j para representar el número de la tabla actual, iniciando en 1.
  3. Usar un ciclo while externo para recorrer las tablas del 1 al 10.
  4. Dentro de este ciclo, definir una variable i para los multiplicadores de cada tabla, iniciando en 1.
  5. Usar un ciclo while interno para imprimir los productos desde j x 1 hasta j x 9.
  6. Separar cada tabla con un salto de línea.

# Tablas de mulltiplicar del 1 al 10
j = 1
while j < 11:
    i = 1
    print("Tabla del ",j)
    while i < 10:
        print(j, "X ", i, " = ", j * i)
        i = i + 1
    print()
    j = j + 1

Actividades sugeridas.

  • Modificar el límite de i para mostrar hasta x 12.
  • Adaptar el código para usar ciclos for en lugar de while.
  • Agregar un separador visual entre tablas, como líneas de guiones.

Experimentos.

  • Pedir al usuario el número inicial y final de tablas a mostrar.
  • Mostrar las tablas en orden inverso, del 10 al 1.
