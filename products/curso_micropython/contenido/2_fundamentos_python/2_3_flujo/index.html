Estructuras de control de flujo

  • Inicio

  • 2.3. Estructuras de control de flujo

Las estructuras de control de flujo en Python permiten decidir qué instrucciones se ejecutan y cuántas veces, dependiendo de condiciones o repeticiones necesarias en el programa. Existen tres tipos principales: secuencia, selección y ciclos.

  1. Estructuras de selección (condicionales)

    Las secuencias son la forma más simple de control de flujo en Python. Consisten en ejecutar instrucciones una tras otra, siguiendo el orden en que aparecen en el programa, de arriba hacia abajo, sin saltos ni repeticiones automáticas.

    Ejemplo: 

    print("Inicio del programa")
x = 5
y = 10
suma = x + y
print("La suma es:", suma)
print("Fin del programa")

    En este caso, cada línea se ejecuta en orden:

    1. Se imprime un mensaje inicial.
    2. Se asignan valores a las variables x y y.
    3. Se calcula la suma.
    4. Se muestra el resultado.
    5. Se imprime el mensaje final.

    Este comportamiento “línea por línea” es la base de cualquier programa, sobre la cual luego se aplican estructuras más avanzadas como condicionales o ciclos.

  2. Estructuras de selección (condicionales)

    La instrucción if-else permite ejecutar un bloque de código según una condición.

    • Si la condición es verdadera, se ejecuta el bloque dentro del if.
    • Si es falsa, se ejecuta el bloque del else.
    • Con elif se pueden evaluar múltiples condiciones.

    Ejemplo básico con if: 

    temperatura = 25
if temperatura > 30:
    print("Hace calor")

    Ejemplo con if - else: 

    if temperatura > 30:
    print("Hace calor")
else:
    print("Temperatura agradable")

    Ejemplo con if - elif - else: 

    if temperatura > 30:
    print("Hace calor")
elif temperatura < 15:
    print("Hace frío")
else:
    print("Temperatura agradable")

  3. Estructuras repetitivas (ciclos)

    1. Bucle For:

      Un ciclo for, estructura ciclo, hace la misma tarea (itera) el número de veces especificado por un iterable (una lista, un string o un rango). 

      for x in range(0, 5, 1):

      • for define el ciclo for.
      • x es la variable que define el número de veces que se ejecuta el bloque de código dentro del ciclo, print(miInt).
      • La función range(start, stop, step) se usa para definir x. El valor inicial se define con start, el valor final se define con stop - 1, y la magnitud con que cambia x en el ciclo se define con step.
      • in es un operador booleano que da True si el valor dado x esta en el rango de una lista, string, rango, etc.

      Ejemplo 1 con ciclo for: 

      for i in range(5):
    print("Número:", i)

      Ejemplo 2 con ciclo for: 

      for i in range(7):
    Led1.on()
    time.sleep(1.5)
    Led1.off()
    time.sleep(1.5)

    2. Bucle While:

      El ciclo while, es una instrucción que itera mientras se cumpla una condición. 

      while x < 5:

      • x la variable que determina si la condición se cumple o no, se define FUERA del encabezado del ciclo while.
      • La condición (x < 5) es una instrucción que produce un valor booleano.

      Ejemplo 1 con ciclo while: 

      contador = 1
while contador <= 5:
    print("Número:", contador)
    contador += 1

      Ejemplo 2 con ciclo while: 

      while True:
    distancia = medir_distancia()
    if distancia < 5:
        activar_zumbador()
        sleep(5)
    else:
        sleep(1)

  4. Control del flujo dentro de ciclos:

    Python dispone de instrucciones especiales:

    • break: sale inmediatamente del ciclo.
    • continue: salta el resto del código de la iteración actual y continúa con la siguiente.

    Ejemplo de control de flujo dentro de ciclos: 

    for i in range(1, 6):
    if i == 3:
        continue  # Salta el número 3
    print(i)

  5. Aplicación en MicroPython con ESP32

    Las estructuras de control son clave para automatizar tareas como:

    • Encender un LED solo si un sensor detecta una condición específica.
    • Repetir lecturas de un sensor hasta alcanzar un valor límite.
    • Ejecutar acciones en intervalos de tiempo.

    Ejemplo de aplicación en ESP32: 

    while True:
    if sensor_distancia.read() < 10:
        encender_led()
    else:
        apagar_led()
