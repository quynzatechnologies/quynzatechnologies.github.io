Módulo 2.4. Funciones y manejo de librerías

  1. Funciones

    Las funciones son bloques de código reutilizables que realizan una tarea específica. Su uso permite:

    • Estructurar los programas en partes más claras.
    • Reducir la repetición de código.
    • Mejorar la legibilidad y el mantenimiento.

    def function(a, b): 
    ...... 
    return a + b

    • def: indica que se está definiendo una nueva función.
    • function() es el nombre de la función. Por convención, este nombre suele ser en minúsculas y representa un verbo/acción.
    • a,b son los parámetros que se utilizan en el bloque de instrucciones de la definición de la función (...). Si la función no tiene parámetros, se utiliza un paréntesis vacío ().
    • La declaración return es una declaración opcional que devuelve el valor de la función a la llamada original.

    Ejemplo básico:
    def saludar(): print("Hola desde la función") saludar()
    Ejemplo con parámetros:
    def sumar(a, b): resultado = a + b return resultado print(sumar(5, 3)) # Imprime 8
    Ejemplo con valor de retorno:
    def convertir_a_fahrenheit(celsius): return (celsius * 9/5) + 32 temp_f = convertir_a_fahrenheit(25) print(temp_f)
  2. Librerías en Python

    Una librería es un archivo que contiene funciones, clases o variables reutilizables.

    • Python incluye una amplia variedad de librerías estándar (por ejemplo, math, os, random).
    • Se pueden crear librerías propias para organizar el código.

    Ejemplo de importación de librerías:
    import math print(math.sqrt(16)) # Raíz cuadrada
    Ejemplo de importación selectiva::
    from math import sqrt print(sqrt(16))
  3. Librerías en Micropython

    En MicroPython, las librerías incluyen control específico de hardware:

    • machine: control de pines, ADC, PWM, I2C, SPI, etc.
    • network: manejo de conexiones WiFi y protocolos de red.

    from machine import Pin

led = Pin(2, Pin.OUT)
led.value(1)  # Encender LED
