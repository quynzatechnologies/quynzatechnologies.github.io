Módulo 2.5. Entrada y salida de datos

En Python y MicroPython, la entrada y salida de datos permiten que un programa interactúe con el usuario o con el entorno físico a través de sensores y actuadores.

  • Entrada de datos: el programa recibe información desde el teclado, sensores u otras fuentes.
  • Salida de datos: el programa muestra o envía información al usuario o a otro sistema.

  1. Entrada desde datos

    1. En Python estándar:

      En Python, se utiliza la función input() para leer datos desde el teclado. Por defecto, devuelve una cadena (str), por lo que a menudo se convierte al tipo adecuado:

      • El parámetro de una función input() es una cadena, string, como: “Entre un número: ".
      • La respuesta al input es una llamada en espera, para que el usuario entre una cadena, string. Para usar este valor, se debe convertir con otra función, por ej., int().

    2. En Micropython:

      En el ESP32, la entrada desde teclado físico no es habitual. La entrada se realiza:

      • Mediante la consola REPL.
      • Por comunicación serie desde un PC u otro dispositivo.
      • A través de sensores conectados al hardware.

  2. Salida de datos en consola

    1. En consola:

      Se usa la función print() para mostrar información en pantalla: 

      print(y)

      • El parámetro de esta función puede ser de cualquier tipo. Todos los tipos se convierten automáticamente a strings.

    2. En hardware (Micropython):

      La salida de datos también puede ser hacia pantallas, LEDs o zumbadores. Por ejemplo, encender un LED para indicar un evento: 

      from machine import Pin

led = Pin(2, Pin.OUT)
led.value(1)  # Encender LED

      O mostrar texto en una pantalla LCD1602: 

      lcd.putstr("Hola Mundo")

  3. Conversión de tipos en entradas

    Al recibir datos como texto, es común transformarlos al tipo que necesitamos:

    valor = float(input("Introduce un valor decimal: ")) print("El doble es:", valor * 2)
