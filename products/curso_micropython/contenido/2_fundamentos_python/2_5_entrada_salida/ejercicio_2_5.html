Ejercicio 2-5: Juego de adivinar número con búsqueda binaria

Objetivo. Adivina un número secreto. Se usa la función de entrada input(), para que el usuario entre un número para ver si es el número secreto. Búsqueda binaria. 

import random

# Genera un número aleatorio de dos dígitos
numero_secreto = random.randint(1, 99)

print("Adivine el número secreto")

intentos = 0
while True:
    entrada = input("Ingrese un número: ")
    
    if not entrada.isdigit():
        print("Entrada no válida. Por favor, ingrese un número.")
        continue # Continua dentro de la estructura if hasta entrar un número válido

    numero_usuario = int(entrada)
    intentos = intentos + 1

    if numero_usuario == numero_secreto:
        print(" ¡Correcto! El número es ", numero_secreto, "fueron",intentos,"intentos")
        break  # Rompe la estructura if y while cuando encuentra el número secreto
    elif numero_usuario < numero_secreto:
        print("El número secreto es mayor.")
    else:
        print("El número secreto es menor.")

Actividades sugeridas.

  • Cambiar el rango del número aleatorio (por ejemplo, entre 1 y 500).
  • Añadir un límite máximo de intentos para incrementar la dificultad.
  • Mostrar un mensaje especial si el usuario acierta en el primer intento.
  • Guardar el historial de intentos en un archivo de texto.
