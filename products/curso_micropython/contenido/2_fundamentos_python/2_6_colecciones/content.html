Módulo 2.6. Manejo de colecciones

En Python, una colección es una estructura que permite almacenar y organizar múltiples valores en una sola variable. Las más utilizadas son: listas, tuplas, diccionarios y cadenas.

  1. Listas (list)

    Son colecciones ordenadas y modificables que pueden contener elementos de diferentes tipos.
    colores = ["rojo", "verde", "azul"] print(colores[0]) # Acceder al primer elemento colores.append("amarillo") # Agregar un elemento print(colores[3]) # Acceder al cuarto elemento
    • Se accede a los elementos mediante índices (comenzando desde 0).
    • Se pueden modificar, añadir o eliminar elementos.
    • Útiles para almacenar series de valores, como lecturas de sensores.

  2. Tuplas (tuple)

    Son colecciones ordenadas pero inmutables.
    dimensiones = (640, 480) print(dimensiones[0])
    • No se pueden modificar después de creadas.
    • Útiles para almacenar datos que no deben cambiar, como configuraciones fijas.

  3. Diccionarios (dict)

    Almacenan datos en forma de clave-valor.
    sensor = {"tipo": "DHT11", "temperatura": 25, "humedad": 60} print(sensor["temperatura"])
    • Permiten acceso rápido a los datos mediante la clave.
    • Son útiles para representar objetos o configuraciones.

  4. Cadenas (str)

    Son secuencias de caracteres que pueden manipularse como listas.
    mensaje = "ESP32" print(mensaje[0]) # 'E' print(len(mensaje)) # Longitud
  5. Operadores y funciones comunes en colecciones

    • in: verifica si un elemento está presente.
    • len(): devuelve el número de elementos.
    • append(): agrega un elemento a una lista.
    • remove(): elimina un elemento de una lista.

    Ejemplo de uso de operadores:
    colores = ["rojo", "verde", "azul"] if "rojo" in colores: print("El color rojo está en la lista") if "negro" in colores: print("El color negro está en la lista")
  6. Aplicación práctica en ESP32

    En MicroPython, las colecciones son muy útiles para:

    • Guardar mediciones de sensores en una lista antes de procesarlas.
    • Almacenar configuraciones en un diccionario para control de dispositivos.
    • Procesar cadenas recibidas desde un puerto serie o por WiFi.
