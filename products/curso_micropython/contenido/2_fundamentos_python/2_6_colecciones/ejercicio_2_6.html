Objetivo. Practicar el uso de estructuras condicionales y ciclos en Python. Aplicar la validación de entradas y el manejo de intentos para un juego interactivo.

Procedimiento.

  1. Abrir Thonny y crear un nuevo archivo .py.
  2. Generar un número aleatorio entre 1 y 99 usando random.randint().
  3. Solicitar al usuario que ingrese un número e implementar validaciones para permitir solo entradas numéricas.
  4. Comparar el número ingresado con el número secreto:
    • Si es correcto, mostrar un mensaje de éxito y el número de intentos realizados.
    • Si es menor, indicar que el número secreto es mayor.
    • Si es mayor, indicar que el número secreto es menor.
  5. 5. Contabilizar el número de intentos hasta que el usuario acierte.

import random

# Lista de 15 posibles menús (puedes cambiar los nombres)
ayerComi = "Nada"
menu_opciones = ["Arroz con pollo","Pasta a la boloñesa","Sopa de verduras", 
    "Cuchuco","Bandeja paisa","Lentejas con chorizo","Pescado al horno",
    "Carne asada","Sobrebarriga","Frijoles con arroz","Pollo a la plancha",
    "Pizza margarita","Hamburguesa casera","Tacos mexicanos","Chuleta de cerdo"]
dias = ["lunes","martes", "miercoles",  "jueves", "viernes"]
i = 0
while i < 5:
    menu_seleccionado = random.choice(menu_opciones)

    if ayerComi == menu_seleccionado:
        print("Seleccione otro menu")
        i = i - 1
    else:   
        # Mostrar el resultado
        print("El menú para el", dias[i], "es:")
        print(menu_seleccionado)
        ayerComi = menu_seleccionado
    i = i + 1

Actividades sugeridas.

  • Cambiar el rango del número aleatorio (por ejemplo, entre 1 y 500).
  • Añadir un límite máximo de intentos para incrementar la dificultad.
  • Mostrar un mensaje especial si el usuario acierta en el primer intento.
  • Modificar el juego para que sea el usuario quien piense el número y el programa intente adivinarlo usando búsqueda binaria.
  • Guardar el historial de intentos en un archivo de texto.
