Ejercicio 2.7. Ejercicios iniciales en Thonny con ESP32

Antes de programar aplicaciones más complejas, es importante realizar ejercicios simples que permitan familiarizarse con el entorno Thonny IDE, la sintaxis de MicroPython y la interacción básica con el ESP32.

  1. Conexión y prueba en Thonny

    1. Conectar el ESP32 al PC mediante cable USB.
    2. En Thonny, ir a Herramientas -> Opciones -> Intérprete.
    3. Seleccionar el puerto correspondiente.
    4. Elegir MicroPython (ESP32) como intérprete.

    En la consola REPL de Thonny: 

    print("Hola desde ESP32")

    Si aparece el mensaje en pantalla, la conexión está funcionando.

  2. Control básico de pines GPIO

    Ejemplo: encender y apagar un LED conectado al pin 2 del ESP32. 
    from machine import Pin
import time

led = Pin(2, Pin.OUT)

while True:
    led.value(1)  # Encender
    time.sleep(1)
    led.value(0)  # Apagar
    time.sleep(1)

    Este código alterna el estado del LED cada segundo, mostrando el uso básico de GPIO, ciclos while y retardos con time.sleep().

  3. Lectura simple desde un sensor

    Ejemplo: leer el estado de un pulsador. 

    from machine import Pin

boton = Pin(4, Pin.IN)

while True:
    if boton.value() == 1:
        print("Botón presionado")
    else:
        print("Botón suelto")
    time.sleep(0.5)

  4. Guardar y ejecutar scripts en el ESP32

    En Thonny:

    1. Escribir el programa en Thonny.
    2. Guardarlo en el dispositivo con el nombre main.py para que se ejecute automáticamente al encenderlo.

Objetivo de estos ejercicios

  • Practicar la escritura y ejecución de código en Thonny.
  • Conocer el ciclo básico de trabajo: escribir, ejecutar, depurar y guardar programas.
  • Familiarizarse con la manipulación de entradas y salidas en el ESP32.
