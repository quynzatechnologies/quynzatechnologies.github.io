Ejercicio 4-1: Control del LED interno del ESP32

  • Ejercicio 4-1: Control del LED interno del ESP32

Objetivo. Aprender a ejecutar un programa en MicroPython con el IDE Thonny y correrlo en el entrenador QMT-020. Comprender el uso básico de los módulos time y machine para el control de hardware. Experimentar con la estructura secuencial y con un bucle infinito (while True).

Procedimiento.

  1. Conectar el ESP32 al computador mediante el cable USB-C.
  2. Abrir el IDE Thonny y preparar un nuevo archivo de programa.
  3. Comprender el uso de los módulos:
    • time: funciones para medir intervalos de tiempo y generar retardos.
    • machine: acceso directo al hardware del microcontrolador, incluyendo la configuración de pines GPIO.
  4. Identificar el pin donde está conectado el LED a controlar.
  5. Para el LED azul integrado en el ESP32, generalmente se usa el GPIO2.
  6. A continuación, vamos a encender y apagar una vez el LED del microcontrolador: 

    # Encendido y apagado único
import time
from machine import Pin

# Crea un objeto pin como LED de salida
led = Pin(2, Pin.OUT) 

led.value(1)      # Enciende el LED
time.sleep(1.5)   # Espera 1.5 segundos
led.value(0)      # Apaga el LED

    Actividad sugerida. Cambie el tiempo en la instrucción time.sleep(1.5) y observe lo que sucede.

  7. Ahora, vamos a prender y a apagar el LED azul en un ciclo infinito. 

    # Encendido y apagado repetitivo (bucle while
import time
from machine import Pin

# Crea un objeto pin como LED de salida
led=Pin(2, Pin.OUT) 

while True:
  led.value(1)      # Enciende el LED
  time.sleep(0.25)
  led.value(0)      # Apaga el LED
  time.sleep(0.25)

    Aquí se utiliza la estructura de control de flujo en la forma while true. Es decir que siempre se va a estar ejecutando el bloque de instrucciones que sigue al while. Observe la indentación del bloque de instrucciones del while.

    Actividad sugerida. Ahora ejecute el programa varias veces cambiando los intervalos de espera.

Experimentos.

  • Observe la ejecución del programa y vea el tiempo de prendido y apagado del LED.
  • Disminuya el tiempo de parpadeo del LED.
  • Aumente el tiempo de parpadeo del LED.
  • Guarde el programa en la carpeta
  • Guarde el programa en el microcontrolador.
