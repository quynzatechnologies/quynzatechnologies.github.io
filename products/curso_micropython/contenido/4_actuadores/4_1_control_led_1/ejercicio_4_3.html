Ejercicio 4-3: Encender y apagar LEDs externos

Objetivo. Practicar la configuración de pines GPIO como salidas. Controlar uno o más LEDs encendiéndolos y apagándolos secuencialmente. Reforzar el uso de instrucciones repetitivas en MicroPython.

Procedimiento.

  1. Conectar un cable desde el pin GPIO26 del microcontrolador al pin del LED azul del entrenador.



  2. Escribir en Thonny el primer bloque de código para encender y apagar el LED azul tres veces.
  3. Ejecutar el código y observar el parpadeo. 

    import time
from machine import Pin

# Configuración del LED azul en GPIO26
led_azul = Pin(26, Pin.OUT)

# Encendido y apagado 3 veces
for i in range(3):
    led_azul.value(1)   # Enciende
    time.sleep(1.5)
    led_azul.value(0)   # Apaga
    time.sleep(1.5)

    Actividad sugerida. Cambiar el número de repeticiones en el ciclo y el tiempo de espera para observar el cambio de comportamiento.

  4. Conectar un segundo cable desde el pin GPIO22 al LED rojo del entrenador.
  5. Escribir en Thonny el segundo bloque de código para encender y apagar ambos LEDs simultáneamente. 

    import time
from machine import Pin

# Configuración de LEDs
led_azul = Pin(26, Pin.OUT)
led_rojo = Pin(22, Pin.OUT)

# Encendido y apagado 3 veces
for i in range(3):
    led_azul.value(1)    # Enciende LED azul
    led_rojo.value(1)    # Enciende LED rojo
    time.sleep(1.5)
    led_azul.value(0)    # Apaga LED azul
    led_rojo.value(0)    # Apaga LED rojo
    time.sleep(1.5)

Actividades sugeridas.

  • Encender los LEDs en secuencia (azul -> rojo) en lugar de simultáneamente.
  • Invertir el orden de encendido y observar los cambios.
