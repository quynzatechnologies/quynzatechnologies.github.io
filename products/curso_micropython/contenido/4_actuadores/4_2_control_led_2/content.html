Módulo 4.2. Manejo de múltiples LEDs y estructuras while/for

El control de múltiples LEDs en un microcontrolador como el ESP32 permite representar estados, secuencias y patrones visuales de forma sencilla y efectiva. Al trabajar con varios LEDs, es necesario comprender cómo manipular varias salidas digitales de manera simultánea o secuencial utilizando estructuras de control repetitivas.

  1. Fundamentos del control múltiple

    En MicroPython, cada LED se controla mediante un GPIO configurado como salida digital. Cuando se manejan varios LEDs, es habitual:

    • Encenderlos simultáneamente para indicar un mismo estado.
    • Encenderlos en secuencia para generar patrones visuales.
    • Usar listas o arreglos para simplificar el código y evitar repetición.

  2. Estructuras repetitivas en MicroPython

    Para controlar LEDs de forma continua o en ciclos controlados se emplean dos estructuras clave:

    • while -> Repite un bloque de código mientras una condición sea verdadera.
      • Ejemplo típico: parpadeo infinito con while True.
    • for -> Itera sobre un rango de valores o elementos de una lista.
      • o Ideal para recorrer varios LEDs almacenados en una lista.

    Estas estructuras permiten automatizar secuencias de encendido/apagado.

  3. Ejemplo básico con varios LEDs 

    from machine import Pin
import time

leds = [Pin(15, Pin.OUT), Pin(16, Pin.OUT), Pin(21, Pin.OUT), Pin(22, Pin.OUT)]

for led in leds:
    led.value(1)    # Encender LED
    time.sleep(0.5)
    led.value(0)    # Apagar LED

  4. Aplicaciones prácticas

    • Indicadores multinivel: mostrar progreso o niveles de señal.
    • Simulación de señales luminosas: intermitentes, alarmas o barridos.
    • Efectos visuales: patrones decorativos o informativos en proyectos IoT.
