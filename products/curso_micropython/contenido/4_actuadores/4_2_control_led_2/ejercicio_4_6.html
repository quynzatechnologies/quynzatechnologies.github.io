Objetivo. Utilizar listas para almacenar referencias a múltiples pines de salida. Implementar un bucle while con variable de control para encender LEDs en secuencia.

Procedimiento.

  1. Conectar un cable desde el pin GPIO15 al LED azul del entrenador.
  2. Conectar el pin GPIO16 al LED verde, el GPIO21 al LED amarillo y el GPIO22 al LED rojo.
  3. Abrir Thonny e identificar que el microcontrolador esté correctamente conectado.
  4. Escribir el código mostrado a continuación en un nuevo archivo.
  5. Ejecutar el programa y observar cómo los LEDs se encienden uno por uno. 

    import time
from machine import Pin
ledAz=Pin(15, Pin.OUT) # Define el pin led as salida
ledV=Pin(16, Pin.OUT)
ledAm=Pin(21, Pin.OUT) # Define el pin led as salida
ledR=Pin(22, Pin.OUT)

lista = [ledAz, ledV,ledAm,ledR]

i = 0               # Variable de control del while
while i < 4:        # Enciende el LED
  led = lista[i]    # Recorre cada LED
  led.value(1)
  time.sleep(0.5)
  led.value(0)      # Apaga el LED
  time.sleep(0.5)
  i = i + 1         # Incrementa la variable de control

Actividades sugeridas.

  • Cambiar el orden de los LEDs en la lista para alterar la secuencia de encendido.
  • Modificar el tiempo de encendido y apagado para observar cambios en la secuencia.

Experimentos.

  • Ampliar la lista para incluir otros dispositivos de salida.
  • Repetir la secuencia indefinidamente usando un bucle while True.
