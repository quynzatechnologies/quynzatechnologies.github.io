Ejercicio 4-7: Barrido secuencial de 4 LEDs con ciclos anidados

Objetivo. Utilizar ciclos while anidados para realizar un barrido de encendido y apagado de varios LEDs. Reforzar el control de secuencias en salidas digitales mediante MicroPython.

Procedimiento.

  1. Conectar el pin GPIO15 al LED azul del entrenador.
  2. Conectar el pin GPIO16 al LED verde, el GPIO21 al LED amarillo y el GPIO22 al LED rojo.
  3. Abrir Thonny y asegurarse de que el ESP32 esté correctamente conectado.
  4. Escribir el código mostrado a continuación en un nuevo archivo.
  5. Ejecutar el programa y observar cómo los LEDs se encienden en secuencia rápida. 

    import time
from machine import Pin
ledAz=Pin(15, Pin.OUT) #Define el pin led azul de salida
ledV=Pin(16, Pin.OUT)  #Define el pin led verde de salida
ledAm=Pin(21, Pin.OUT) #Define el pin led amarillo de salida
ledR=Pin(22, Pin.OUT)  #Define el pin led rojo de salida

lista = [ledAz, ledV,ledAm,ledR]
while True:
    i = 0
    while i < 4:      
      led = lista[i]   # Enciende el LED sub i
      led.value(1)
      time.sleep(0.07)
      led.value(0)     # Apaga el LED sub i
      time.sleep(0.07)
      i = i + 1

Actividades sugeridas.

  • Cambiar el tiempo de espera para ralentizar o acelerar el barrido.
  • Alterar el orden de los LEDs en la lista para modificar la secuencia.

Experimentos.

  • Hacer que el barrido sea bidireccional (ida y vuelta).
  • Probar el barrido con diferentes combinaciones de colores de LEDs.
