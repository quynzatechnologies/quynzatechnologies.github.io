Ejercicio 4-10: Mostrar texto fijo en pantalla LCD1602

Objetivo. Aprender a utilizar la pantalla LCD1602 para mostrar mensajes de texto. Mostrar letreros fijos en una o dos líneas de la pantalla.

Procedimiento.

  1. Conectar el pin GPIO22 del ESP32 al pin SCL del módulo LCD1602 y el pin GPIO21 al pin SDA.



  2. Abrir Thonny e importar la librería lcd1602, que ya incluye la configuración de pines SDA y SCL.
  3. Escribir el código mostrado a continuación para mostrar dos mensajes fijos en la pantalla.
  4. Ejecutar el programa y observar cómo se muestran los mensajes uno tras otro. 
    #LCD_1602
from lcd1602 import LCD
import time

# Crea una instancia de la clase LCD y le asigna la variable lcd 
lcd = LCD()
# Define el string " QUYNZA\n"

string = " QUYNZA\n"
# Presenta el string en la pantalla
lcd.message(string)

time.sleep(2)
# Define el string "    Tecnologia!""
string = "    Tecnologia!"
# Presenta el string en la pantalla
lcd.message(string)

time.sleep(2)
# Limpia la pantalla 
lcd.clear()

Actividades sugeridas.

  • Cambiar los letreros mostrados.
  • Mostrar un letrero que se vaya formando letra por letra.
  • Mostrar un letrero que se desplace horizontalmente en la pantalla.
  • Guardar el programa en la carpeta local del computador.
  • Guardar el programa directamente en el microcontrolador y ejecutarlo sin conexión al PC.
