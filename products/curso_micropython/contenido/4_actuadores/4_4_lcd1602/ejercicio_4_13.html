Ejercicio 4-13: Concatenar texto y variables en LCD1602

Objetivo. Mostrar en pantalla LCD1602 mensajes que combinen texto fijo y valores almacenados en variables tipo string. Reforzar el uso de la concatenación de cadenas en MicroPython.

Procedimiento.

  1. Conectar el pin GPIO22 del ESP32 al pin SCL del módulo LCD1602 y el pin GPIO21 al pin SDA.



  2. Abrir Thonny e importar la librería lcd1602.
  3. Escribir el código proporcionado que toma elementos de una lista y los concatena con un texto para mostrarlos en pantalla.
  4. Ejecutar el programa y observar que el mensaje se muestra con cada nombre de la lista. 

    from lcd1602 import LCD
import time

# Crea una instancia de la clase LCD y le asigna la variable lcd 
lcd = LCD()

lista = ["Juan", "Luis", "Maria", "Clara", "Pedro"]
i = 0
while i < 5:
    nombre = lista[i]       # Variable  tipo string
    mensaje = "Hoy, "+ nombre + " feliz"
    lcd.message(mensaje)    # Se suman los strings
    time.sleep(2)
    lcd.clear()
    i = i + 1
lcd.clear()

Actividades sugeridas.

  • Cambiar el mensaje de forma que incluya datos numéricos convertidos a string.
  • Añadir una segunda línea con información adicional para cada nombre.
  • Mostrar los mensajes en orden inverso al de la lista.
  • Guardar el programa en la carpeta local del computador.
  • Guardar el programa directamente en el microcontrolador y ejecutarlo sin conexión al PC.
