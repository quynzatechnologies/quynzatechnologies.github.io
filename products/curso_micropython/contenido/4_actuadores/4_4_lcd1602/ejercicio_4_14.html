Ejercicio 4-14: Elegir ganador aleatorio y mostrar en LCD1602

Objetivo. Seleccionar aleatoriamente un elemento de una lista usando la librería random. Mostrar el elemento seleccionado en una pantalla LCD1602.

Procedimiento.

  1. Conectar el pin GPIO22 del ESP32 al pin SCL del módulo LCD1602 y el pin GPIO21 al pin SDA.



  2. Abrir Thonny e importar la librería lcd1602 y random.
  3. Crear una lista de nombres y usar el método randint para generar un número aleatorio que sirva como índice.
  4. Mostrar el nombre seleccionado en la pantalla LCD1602. 

    from lcd1602 import LCD
import time
import random               # Libreria de números aleatorios

lcd = LCD()

lista = ["Juan", "Luis", "Maria", "Clara", "Pedro", "Vilma"]
dado = random.randint(1, 6) # Genera un numero aleatorio entre 1 y 6
dado = dado - 1
nombre = lista[dado]        # Las listas comienzan con indice 0
lcd.message("Gana "+ nombre)

Actividades sugeridas.

  • Cambiar la lista por nombres de equipos o proyectos.
  • Mostrar el ganador junto con la hora actual usando un módulo RTC.
  • Repetir la selección automáticamente cada cierto intervalo de tiempo.
  • Guardar el programa en la carpeta local del computador.
  • Guardar el programa directamente en el microcontrolador y ejecutarlo sin conexión al PC.
