Objetivo. Generar un número aleatorio con la librería random. Convertir un entero a cadena de texto para mostrarlo en la pantalla LCD1602.

Procedimiento.

  1. Conectar el pin GPIO22 del ESP32 al pin SCL del módulo LCD1602 y el pin GPIO21 al pin SDA.



  2. Abrir Thonny e importar la librería lcd1602 junto con random para la generación de números aleatorios.
  3. Crear una variable dado que contenga un número entero aleatorio entre 1 y 6.
  4. Convertir el número entero en cadena con str() para mostrarlo en la pantalla. 

    from lcd1602 import LCD
import time
import random               # Libreria de números aleatorios

lcd = LCD()

dado = random.randint(1, 6) # Genera un numero aleatorio entre 1 y 6

dadoStr = str(dado)         # Conversión de entero a string
lcd.message("Salio el "+ dadoStr)

Actividades sugeridas.

  • Repetir el lanzamiento del dado cada 5 segundos y mostrar el resultado en la pantalla.
  • Mostrar el resultado del dado junto con un mensaje motivacional.
  • Modificar el rango para simular un dado de 12 caras.
  • Guardar el programa en la carpeta local del computador.
  • Guardar el programa directamente en el microcontrolador y ejecutarlo sin conexión al PC.
