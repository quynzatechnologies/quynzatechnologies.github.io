Ejercicio 4-16: Generar y mostrar número de lotería con formato en LCD1602

  • Ejercicio 4-16: Generar y mostrar número de lotería con formato en LCD1602

Objetivo. Generar un número aleatorio con la librería random. Aplicar la estructura de control if para formatear el número y mantener los ceros a la izquierda. Mostrar el resultado en la pantalla LCD1602.

Procedimiento.

  1. Conectar el pin GPIO22 del ESP32 al pin SCL del módulo LCD1602 y el pin GPIO21 al pin SDA.



  2. Abrir Thonny e importar la librería lcd1602 junto con random para la generación de números aleatorios.
  3. Crear una variable que contenga un número entero aleatorio entre 0 y 9999.
  4. Convertir el número en cadena y, si es necesario, añadir ceros a la izquierda usando if y elif.
  5. Mostrar el mensaje y el número en la pantalla LCD1602. 

    from lcd1602 import LCD
import time
import random

lcd = LCD()

numero_loteria = random.randint(0, 9999)
numero = str(numero_loteria) # Convierte a string
if numero_loteria < 10:
    numero = " 000" + numero # Se pegan 3 ceros
elif numero_loteria < 100:
    numero = " 00" + numero  # Se pegan dos ceros
elif numero_loteria < 1000:
    numero = " 0" + numero   # Se pega un cero
else:
    numero = numero
    
lcd.message("Gane la loteria \n")
lcd.message("Compre el "+ numero)

Actividades sugeridas.

  • Mostrar el número de lotería junto con la fecha actual usando un módulo RTC.
  • Repetir el sorteo cada cierto intervalo de tiempo.
  • Cambiar el formato para que el número aparezca centrado en la pantalla.
  • Guardar el programa en la carpeta local del computador.
  • Guardar el programa directamente en el microcontrolador y ejecutarlo sin conexión al PC.
