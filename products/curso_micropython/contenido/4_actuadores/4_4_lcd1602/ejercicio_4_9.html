Ejercicio 4-9: Mostrar lista de nombres en pantalla LCD1602

Objetivo. Conectar y configurar una pantalla LCD1602 con el ESP32. Mostrar secuencialmente elementos de una lista en la pantalla LCD. Utilizar la librería lcd1602 desarrollada por Quynza para manejar la pantalla.

Procedimiento.

  1. Conectar el pin GPIO22 del ESP32 al pin SCL del módulo LCD1602 y el pin GPIO21 al pin SDA.



  2. Abrir Thonny e importar la librería lcd1602 que ya está almacenada en la memoria del ESP32.
  3. Escribir el código proporcionado, que recorre una lista de nombres y los muestra uno por uno en la pantalla.
  4. Ejecutar el programa y verificar que los nombres aparezcan durante 2 segundos antes de borrarse. 

    from lcd1602 import LCD
import time

# Crea una instancia de la clase LCD y le asigna la variable lcd 
lcd = LCD()

lista = ["Juan", "Luis", "María", "Clara", "Pedro"]
i = 0
while i < 5:
    nombre = lista[i]
    lcd.message(nombre)
    time.sleep(2)
    lcd.clear()
    i = i + 1
lcd.clear()   # Limpia la pantalla LCD

Actividades sugeridas.

  • Cambiar la lista de nombres por datos obtenidos de un sensor.
  • Aumentar o disminuir el tiempo de visualización de cada nombre.
  • Mostrar los nombres en una segunda línea con información adicional.
  • Guardar el programa en la carpeta local del computador.
  • Guardar el programa directamente en el microcontrolador y ejecutarlo sin conexión al PC.
