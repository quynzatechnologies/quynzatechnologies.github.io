  • 4.5. Uso del zumbador

El zumbador (buzzer) es un componente electrónico capaz de generar sonido al aplicar una señal eléctrica. Se emplea ampliamente en sistemas de alerta, indicación sonora y reproducción de melodías simples.

En el entrenador QMT-020, el zumbador está conectado a un pin GPIO del ESP32, lo que permite controlarlo mediante señales digitales o moduladas por ancho de pulso (PWM).

  1. Tipos de zumbadores

    • Pasivos: requieren una señal alterna para generar sonido, como la que se obtiene mediante PWM. Son ideales para reproducir tonos variables y melodías.
    • Activos: integran un oscilador interno y emiten un tono fijo al aplicarles tensión continua.

    En este curso utilizaremos zumbadores activo, ya que permiten mayor control desde el código.

  2. Formas de control

    1. Tono fijo -> Se enciende y apaga el zumbador en intervalos regulares. Útil para alarmas simples.
    2. Melodías -> Se usan frecuencias específicas para cada nota musical, generando secuencias de sonidos reconocibles.
    3. Alarmas condicionadas -> El zumbador se activa solo cuando se cumple una condición (por ejemplo, un valor de sensor supera un umbral).

  3. Control digital vs PWM

    • Control digital simple: el GPIO envía un nivel alto (1) para encender y bajo (0) para apagar el zumbador.
    • Control PWM: mediante la modulación del ciclo de trabajo y la frecuencia, se pueden variar la altura (pitch) y la intensidad del sonido.

  4. Aplicaciones prácticas

    • Alarmas por temperatura, distancia u otros parámetros críticos.
    • Indicadores acústicos en sistemas de control.
    • Reproducción de melodías como alertas personalizadas.
    • Interfaces hombre-máquina con retroalimentación sonora.
