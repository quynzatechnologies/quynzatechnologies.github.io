Ejercicio 5-2: Lectura de fotorresistencia con visualización en LCD

Objetivo. Mostrar en pantalla LCD1602 el valor leído por la fotorresistencia. Reforzar el uso combinado de entradas analógicas y salidas en LCD.

Procedimiento.

  1. Conectar la fotorresistencia al pin GPIO36 del ESP32.

  2. Conectar la pantalla LCD1602 al bus I2C del ESP32.

  3. Abrir Thonny y crear un nuevo archivo .py.
  4. Configurar el pin de la fotorresistencia como entrada analógica (ADC).
  5. Configurar la atenuación ADC a 11dB para leer el rango completo de valores.
  6. Crear una instancia de LCD y mostrar el valor leído como cadena de texto. 

    from machine import ADC,Pin
import time
from lcd1602 import LCD

foto_resistencia = ADC(Pin(36, Pin.IN))

# Configura la atenuación ADC a 11dB para un rango completo
foto_resistencia.atten(foto_resistencia.ATTN_11DB)
while True:

    # Lee un valor análogo bruto entre 0-4095
    valor = foto_resistencia.read()
    # A mayor iluminación, menor resistencia
    print(valor)
    time.sleep(0.05)
    string = str(valor)
    time.sleep(0.5)
    lcd = LCD()
# Presenta el string en la pantalla
    lcd.message(string)

Actividades sugeridas.

  • Observar los valores en la pantalla en diferentes condiciones de luz.
  • Acercar una luz intensa y verificar el cambio en tiempo real.
  • Cubrir la fotorresistencia para ver cómo aumenta la lectura.
  • Cambiar el intervalo de actualización modificando time.sleep(0.5).
  • Guardar el programa en el microcontrolador y ejecutarlo sin conexión al PC.
