Módulo 5.7. Lectura de control remoto IR

Los controles remotos infrarrojos (IR) son dispositivos de transmisión de datos a corta distancia que utilizan luz infrarroja modulada para enviar comandos a un receptor. El receptor IR decodifica las señales luminosas y las convierte en información digital que el microcontrolador puede interpretar.

  1. Principio de funcionamiento

    • La comunicación IR requiere línea de vista directa entre el control y el receptor.
    • No atraviesa paredes ni objetos opacos, a diferencia de tecnologías como WiFi o Bluetooth.
    • La transmisión se realiza mediante modulación de un haz de luz IR en una frecuencia portadora (típicamente 38 kHz).
    • Cada botón del control envía un código único que representa la función asociada.

  2. Protocolo NEC

    Uno de los protocolos más usados en controles IR es el NEC, que define cómo se codifican y transmiten los datos.

    • Bits de dirección: identifican el dispositivo.
    • Bits de comando: indican la acción solicitada.
    • Bits de control: para repetición o validación de datos.

    En este curso se usará la librería ir_rx.nec.NEC_8 de MicroPython, que implementa la recepción y decodificación del protocolo NEC.

  3. Conexión al ESP32

    En el entrenador QMT-020:

    • VCC: 3.3V o 5V (según módulo).
    • GND: Tierra.
    • OUT: señal digital conectada, por ejemplo, al GPIO15.

  4. Lectura en MicroPython

    El flujo básico es:

    1. Conectar el receptor IR al pin correspondiente.
    2. Usar la librería NEC_8 para decodificar los códigos recibidos.
    3. Implementar una función decodeKeyValue() que convierta cada código en el nombre de la tecla.
    4. Definir un callback() para manejar la tecla presionada y ejecutar la acción correspondiente.

    Ejemplo simplificado: 

    from machine import Pin
from ir_rx.nec import NEC_8
from ir_rx.print_error import print_error

def decodeKeyValue(data):
    if data == 0x16: return "0"
    if data == 0x0C: return "1"
    return "ERROR"

def callback(data, addr, ctrl):
    if data >= 0:
        print(decodeKeyValue(data))

ir = NEC_8(Pin(15, Pin.IN), callback)
ir.error_function(print_error)

while True:
    pass

  5. Aplicaciones

    • Control remoto de LEDs y actuadores.
    • Menús interactivos en pantallas LCD.
    • Control de robots o sistemas de domótica.
    • Ejecución de operaciones programadas (por ejemplo, calculadora).
