Ejercicio 5-11: Recepción y decodificación de señales de control remoto IR

Objetivo. Aprender a recibir y decodificar señales de un control remoto IR usando el protocolo NEC. Interpretar y mostrar en consola la tecla pulsada en el control.

Procedimiento.

  1. Conectar el receptor IR al pin GPIO15 del ESP32.



  2. Abrir Thonny y crear un nuevo archivo .py.
  3. Importar las librerías necesarias: Pin, NEC_8 y print_error.
  4. Implementar la función decodeKeyValue() para traducir el código recibido al nombre de la tecla.
  5. Definir la función callback() para capturar y mostrar en consola la tecla presionada.
  6. Instanciar el receptor NEC_8 y activar la función de manejo de errores.
  7. Ejecutar el ciclo principal para mantener el script activo hasta interrupción manual. 

    import time
from machine import Pin, freq
from ir_rx.print_error import print_error
from ir_rx.nec import NEC_8

# GPIO15 receptor IR
pin_ir = Pin(15, Pin.IN)

# Decodifica los datos recibidos y devuelve el nombre de la tecla correspondiente
def decodeKeyValue(data):
    if data == 0x16:
        return "0"
    if data == 0x0C:
        return "1"
    if data == 0x18:
        return "2"
    if data == 0x5E:
        return "3"
    if data == 0x08:
        return "4"
    if data == 0x1C:
        return "5"
    if data == 0x5A:
        return "6"
    if data == 0x42:
        return "7"
    if data == 0x52:
        return "8"
    if data == 0x4A:
        return "9"
    if data == 0x09:
        return "+"
    if data == 0x15:
        return "-"
    if data == 0x7:
        return "EQ"
    if data == 0x0D:
        return "U/SD"
    if data == 0x19:
        return "CYCLE"
    if data == 0x44:
        return "PLAY/PAUSE"
    if data == 0x43:
        return "FORWARD"
    if data == 0x40:
        return "BACKWARD"
    if data == 0x45:
        return "POWER"
    if data == 0x47:
        return "MUTE"
    if data == 0x46:
        return "MODE"
    return "ERROR"

# El usuario oprime una tecla 
def callback(data, addr, ctrl):
    if data < 0:  # El protocolo NEC envía un código.
        pass
    else:
        print(decodeKeyValue(data))
        # Decodifica e imprime el código

ir = NEC_8(pin_ir, callback) # Instancia el receptor NEC_8 

# Muestra información de depuración, en caso de error
ir.error_function(print_error)

# Mantiene corriendo el script hasta que se interrupe al orpimir el control 
try:
    while True:
        pass
except KeyboardInterrupt:
    ir.close()  # Cierra el receptor
