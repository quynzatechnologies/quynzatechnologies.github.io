Módulo 6.3. Control de LEDs RGB con pulsadores

Los LEDs RGB (Red, Green, Blue) permiten generar una amplia gama de colores combinando diferentes intensidades de los tres colores básicos. Estos componentes son muy comunes en proyectos electrónicos y se controlan ajustando la señal de cada uno de sus pines.

En el contexto del ESP32 y el entrenador QMT-020, es posible manejar un LED RGB utilizando pulsadores para cambiar su color de manera interactiva, lo que permite comprender conceptos básicos de entradas digitales, salidas digitales y lógica condicional.

  1. Funcionamiento básico

    1. Un LED RGB tiene tres diodos internos (rojo, verde y azul) encapsulados en un mismo cuerpo.
    2. Existen dos tipos: ánodo común y cátodo común, que determinan cómo se conecta la alimentación y el control.
    3. Encendiendo cada diodo por separado se obtiene su color puro; combinando dos o más, se generan colores adicionales.

  2. Integración con pulsadores

    En este ejercicio, se emplean dos pulsadores para seleccionar el color:

    • Pulsador 1 presionado: el LED muestra color rojo.
    • Pulsador 2 presionado: el LED muestra color verde.
    • Ambos pulsadores presionados: el LED muestra color azul.
    • Ninguno presionado: el LED permanece apagado.

    Este método introduce la lógica booleana y el concepto de combinaciones de entradas para controlar salidas.

  3. Conexiones en el QMT-020

    • Pulsador 1: GPIO27.
    • Pulsador 2: GPIO18.
    • LED RGB:
      • Rojo -> GPIO14.
      • Verde -> GPIO12.
      • Azul -> GPIO13.

    Cada pin del LED RGB debe tener su resistencia limitadora para evitar sobrecorriente.

  4. Aplicaciones

    • Interfaz básica para seleccionar modos o estados en un sistema.
    • Visualización de alarmas según la condición del sistema (colores predefinidos).
    • Proyectos educativos para explicar lógica digital y codificación binaria.
