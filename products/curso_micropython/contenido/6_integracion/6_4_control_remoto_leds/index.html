Control remoto IR para manejo de LEDs y otras salidas

  • 6.4. Control remoto IR para manejo de LEDs y otras salidas

Los controles remotos por infrarrojo (IR) son una tecnología ampliamente utilizada para enviar señales inalámbricas a corta distancia. Funcionan mediante la emisión de pulsos de luz infrarroja modulados a una frecuencia específica (generalmente 38 kHz) que son recibidos y decodificados por un sensor IR.

En el ESP32, esta tecnología puede integrarse fácilmente para controlar actuadores como LEDs, zumbadores o relés, permitiendo interacción remota sin necesidad de conexión física.

  1. Principio de funcionamiento

    • Emisor (control remoto): genera una secuencia de pulsos que codifican una orden (por ejemplo, encender un LED).
    • Receptor IR: detecta la señal, la demodula y entrega un código digital único al microcontrolador.
    • ESP32: interpreta el código recibido y ejecuta la acción programada (encender, apagar, cambiar color, activar zumbador, etc.).

  2. Integración en el QMT-020

    El módulo receptor IR se conecta normalmente a un GPIO configurado como entrada digital, mientras que las salidas (LEDs, zumbador, etc.) se conectan a pines de salida.

    Ejemplo de asignaciones:

    • Receptor IR -> GPIO15.
    • LED rojo -> GPIO26.
    • LED verde -> GPIO27.
    • Zumbador -> GPIO19.

  3. Decodificación y asignación de funciones

    En MicroPython se utilizan librerías que permiten:

    1. Capturar la señal IR recibida.
    2. Traducirla a un código hexadecimal.
    3. Asociar cada código a una función específica:
      • Encender/apagar un LED.
      • Cambiar de color en un LED RGB.
      • Activar/desactivar un zumbador.
      • Ejecutar una secuencia de acciones.

  4. Aplicaciones prácticas

    • Encender y apagar luces de forma remota.
    • Controlar el volumen o estado de un sistema de sonido.
    • Crear interfaces de usuario simplificadas para proyectos IoT.
    • Implementar sistemas de control de múltiples salidas desde un solo control remoto.
