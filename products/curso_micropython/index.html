Contenido Destacado

Bases de Programación en Python

Aprende los fundamentos de Python, el lenguaje utilizado para programar en Micropython, y construye una base sólida de conocimientos.

Programación con Micropython

Domina la programación del microcontrolador ESP32 usando Micropython, adecuado para proyectos mecatrónicos.

Sensores y Actuadores

Conoce cómo conectar y controlar sensores y actuadores, como LEDs, botones y sensores de temperatura, para crear proyectos interactivos.

Experimentos Prácticos

Realiza ejercicios prácticos que te permitirán comprender cómo funciona la programación del ESP32 con ejemplos reales.

Proyecto Final

Aplica todo lo aprendido en un proyecto final, donde desarrollarás una solución práctica y completa usando el ESP32 y Micropython.

Beneficios para los Participantes

Aplicación Inmediata

Comienza a desarrollar tus propios proyectos mecatrónicos y experimentar con el ESP32 desde el primer día del curso.
